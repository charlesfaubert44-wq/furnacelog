## Task Library Data Structure

Each task contains:

```javascript
{
  // Basic Information
  name: "Task Name",
  description: "Detailed description with northern context",
  category: "routine | seasonal | reactive | emergency",

  // Applicability
  applicableSystems: ["heating", "plumbing", "ventilation"],
  applicableHomeTypes: ["modular", "stick-built", "log", "mobile"],

  // Scheduling
  scheduling: {
    intervalDays: 30,                    // Days between occurrences
    seasonal: {
      applicable: true,
      seasons: ["pre-freeze-up", "winter"]
    },
    triggerConditions: [                 // Event-based triggers
      "After heavy snowfall",
      "When temperature drops below -30°C"
    ]
  },

  // Execution Details
  execution: {
    difficultyLevel: "diy-easy | diy-moderate | professional",
    estimatedMinutes: 30,
    toolsRequired: ["Screwdriver", "Flashlight"],
    suppliesRequired: ["Replacement filter", "Cleaning solution"],
    instructions: [
      "Step 1: Turn off power",
      "Step 2: Remove panel",
      // ... detailed steps
    ],
    safetyWarnings: [
      "Always turn off power before servicing",
      "Wear safety glasses"
    ],
    videoUrl: "https://youtube.com/..."  // Optional tutorial
  },

  // Cost Estimates
  cost: {
    diyEstimate: 15,           // Parts only
    professionalEstimate: 100   // Parts + labor
  },

  // Relationships
  relatedTasks: [ObjectId],    // Follow-up or prerequisite tasks
  isBuiltIn: true,             // System task vs user-created
  createdBy: ObjectId          // null for built-in
}
```

## Northern-Specific Features in Task Library

### 1. Extreme Cold Considerations
**Temperature triggers:**
- "-30°C: Increase HRV filter checks"
- "-40°C: Check heat trace daily"
- "-50°C: Emergency protocols"

**Cold-specific tasks:**
- Ice accumulation monitoring
- Freeze event logging
- Extreme cold heat trace checks

---

### 2. Extended Heating Season
**Impact on maintenance:**
- Filters need more frequent changes (monthly vs quarterly)
- Furnaces run 8-10 months (vs 4-6 months south)
- Combustion analysis critical (extended use)

**Season-adjusted intervals:**
- Summer: Same as south
- Winter: 2-3x more frequent than south

---

### 3. Snow and Ice Management
**Tasks include:**
- Venting system snow clearance
- HRV hood ice removal
- Heat trace activation timing
- Condensate drain freeze prevention

**After-snowfall triggers:**
- Inspect all exterior vents
- Check HRV operation
- Verify heat trace active

---

### 4. Remote Community Considerations
**Task descriptions emphasize:**
- DIY capability when possible
- Order supplies in advance
- Seasonal delivery limitations
- Emergency preparedness

**Cost awareness:**
- Parts cost 1.5-3x higher than south
- Professional labor $150-300/hour
- Fly-in service premiums

---

### 5. Safety Warnings
**Northern-specific hazards:**
- Carbon monoxide (tight building envelopes)
- Freeze damage (hours, not days)
- Electrical (cold affects connections)
- Fuel handling (oil gelling, propane pressure)

**Every task includes:**
- Relevant safety warnings
- When to call professional
- Emergency procedures
